Abstract

Inclusion Net is a peer-to-peer (P2P) lending platform that connects borrowers and lenders directly, removing traditional intermediaries. It uses ML-based credit scoring, KYC verification, real-time chat, intelligent loan matching, and an admin dashboard for platform oversight. By leveraging dynamic algorithms, the platform establishes a transparent, efficient, and accessible financial ecosystem tailored to individual needs.

Objective

  • To develop a travel planning application based on user preferences.
  • To implement ML-based credit scoring for accurate creditworthiness assessment and risk mitigation.
  • To provide intelligent loan matching based on lender preferences and borrower profiles.
  • To improve financial inclusion through accessible technology.

Proposed Solution

Inclusion Net provides automated credit scoring, FOIR-based loan eligibility calculation, intelligent matching algorithms, and real-time WhatsApp-like chat. An admin dashboard manages KYC approvals, loan reviews, and analytics. The platform enforces strict role-based access control utilizing robust JWT authentication mechanisms to ensure comprehensive data security.
