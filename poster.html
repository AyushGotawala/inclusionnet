Project Abstract

Redefining Financial Intermediation

Inclusion Net is a peer-to-peer (P2P) lending platform that connects borrowers and lenders directly, removing traditional intermediaries. It uses ML-based credit scoring, KYC verification, real-time chat, and intelligent loan matching.

The Proposed Solution

Inclusion Net provides automated credit scoring, FOIR-based loan eligibility calculation, intelligent matching algorithms, and real-time WhatsApp-like chat. An admin dashboard manages KYC approvals, loan reviews, and analytics. The platform enforces role-based access control using JWT authentication.
